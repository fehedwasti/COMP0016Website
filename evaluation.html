Critical Evaluation



User Interface
 The VR game provides an interactive and clear user interface that allows users to easily select the settings for their boxing session. They have the ability to change the difficulty which adjusts the speed and spawn rate of the targets that approach the user.
Users can navigate through the main menu using the Mixed Reality joysticks and can also go through a simple read-through guide to understand how to play the game.

Functionality
 The Skeletal tracking library can be used on all available operating systems. For testing purposes, developers need a working version of Windows to use the Kinect SDK. MacOS users can install Windows using Bootcamp for testing their software. The VR Boxing game however is only compatible with computers that can use Windows Mixed Reality Headsets. The project can easily be rebuilt to work with other headsets such as the HTC Vive but will only work on Windows.
The navigation around the Main Menu is very smooth and the loading time is negligable so users will have a continuous and uninterrupted experience.

Stability
 The game works well on compatible machines (mentioned above). There are no known bugs and no instances that cause the game to crash.
As for the skeletal tracking library, the accuracy of the sensor detection improves as more and more videos are recorded allowing the sensor to train and distinguish different actions. As a result, the accuracy solely relies on how many recordings the developer creates.

Project Management
 As a team we deciced to split up important tasks amonst each other. This way, we would be able to finish tasks in a more efficient and quicker manner. Any tasks we felt would take a lot longer or would require new skills were worked on together as a team. Biweekly reports and weekly meetings were used to make sure our team was reaching the weekly goals set at the start of each week.

Future Work



Our team managed to successfully implement all of the Must have and Should have requirements however there were still several improvements we wanted to have a look at. We look to implement online multiplayer features for the game. The idea of a Global Leaderboard would make the game more competetive rather than just a tool for training your boxing skills. We also want to allow users to spar with their friends online which would make the game much more engaging and further improve its appeal to all users. Additionally, if we had more time, we would've looked to implement training practice for another sport/activity. Golf was a very popular activity which our personas found interesting and the ability to practice multiple sports would've made the game even more engaging and usefule to the players.
