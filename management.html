Management

Legal Issues



Potential Liability
Personas that are looking to improve their boxing techniques should not solely rely on the game to advance their skills and should continue training as they normally would alongside with the project. The game is not licensed to be a proper training tool and is meant for basic personal training and entertainment.

Use of Virtual Reality headsets prevents vision of real-life surroundings. Being unaware of the surrounding environment during a physical activity can result in serious injury and/or damage to property. The game is not liable for any of these damages and so users must use the product in a clear space and remove all potential hazards and objects

Data Privacy
Users of the boxing game are recorded for comparison within the sensor to determine which actions the users are performing. These recordings are not stored anywhere and are immediately deleted once the action is determined. No personal data is stored by the application.



User Manual



User Manual
For the skeltal tracking library, developers need to record videos of the actions they want to train the sensor to detect. Once the sensor is able to detect and distinguish different actions at a high success rate, then the tracker can be implemented into any application. The sensor will return different integer values corresponding to the action detected so depending on what action was detected, the application can respond accordingly to meet the requirements of the developer.

As for the VR Boxing game, users must have a NVIDIA GTX 965M graphics card or greater, running Windows 10 and a minimum of 8GB of RAM. Users also need to have the latest version of the Kinect SDK installed on their computer to allow the sensor to be used. To play the game, the user just needs to connect their VR Headset and their Kinect Sensor to their computer and run the executable file which if not found can be built from the Unity Project. Once in the game, the main menu is very easy to navigate, and users should be able to start their sessions without any issues.

Deployment Manual
In order to build the VR Boxing game from the source code, open the Unity Project with the latest version of Unity. Go to File>Build Settings select PC, Mac & Linux standalone. Make sure the Target Platform is Windows and click Build. This should produce the executable file for the VR Boxing Game.
